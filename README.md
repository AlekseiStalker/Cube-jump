# Cube-jump

This game is a typical arcade game where the player is cube, which must jump over the obstacles. 
## Game features 
- Store of different cubes
- Table of best results
- Sound settings
- 9 unique levels
- After 5 level gravitation will change

### Control key: jump [Space]
